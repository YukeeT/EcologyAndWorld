# Экология и мир

Информационный ресурс о заповедниках и охраняемых природных территориях Республики Беларусь.

## Описание проекта

Веб-приложение для изучения заповедников Беларуси через интерактивные карты и подробную информацию о каждом объекте. Проект создан с использованием React для frontend и Python Flask для backend.

## Структура проекта

```
EcologyAndWorld/
├── backend/              # Python Flask backend
│   └── app.py           # Основной файл API
```

## Установка и запуск

### Требования

- Node.js (версия 16 или выше)
- Python 3.8 или выше
- npm или yarn

### Установка зависимостей

#### Frontend (React)

```bash
npm install
```

#### Backend (Python)

```bash
pip install -r requirements.txt
```

### Добавление изображений

Перед запуском приложения необходимо добавить изображения в соответствующие папки:

1. **Главная секция (Hero):**
   - Поместите фоновое изображение в `public/images/hero/hero-background.jpg`
   - Рекомендуемый размер: 1920x1080px или больше

2. **Секция "О нас":**
   - Поместите изображение в `public/images/about/about-nature.jpg`
   - Рекомендуемый размер: 800x600px или больше

3. **Заповедники:**
   - Поместите изображения заповедников в `public/images/reserves/`:
     - `belovezhskaya-pushcha.jpg`
     - `berezinsky.jpg`
     - `pripyatsky.jpg`
     - `narochansky.jpg`
     - `braslavskie.jpg`
     - `polesky.jpg`
   - Рекомендуемый размер: 1200x800px или больше

4. **Секция контактов:**
   - Поместите изображение в `public/images/contact/contact-nature.jpg`
   - Рекомендуемый размер: 800x600px или больше

### Запуск приложения

#### 1. Запуск backend сервера

Откройте терминал и выполните:

```bash
cd backend
python app.py
```

Backend будет доступен по адресу: `http://localhost:5000`

#### 2. Запуск frontend приложения

Откройте новый терминал и выполните:

```bash
npm start
```

Frontend будет доступен по адресу: `http://localhost:3000`

## Использование

1. Откройте браузер и перейдите на `http://localhost:3000`
2. Используйте навигационное меню для перехода между разделами
3. В разделе "Заповедники" вы можете:
   - Просматривать список всех заповедников
   - Кликать на карточки для просмотра детальной информации
   - Использовать интерактивную карту для изучения расположения заповедников
   - Кликать на маркеры на карте для получения информации

## API Endpoints

### GET /api/reserves
Возвращает список всех заповедников Беларуси.

**Ответ:**
```json
[
  {
    "id": 1,
    "name": "Беловежская пуща",
    "description": "...",
    "area": "150000",
    "location": {
      "lat": 52.5736,
      "lng": 23.7994
    },
    "established": "1939",
    "image": "/images/reserves/belovezhskaya-pushcha.jpg",
    "features": ["Бизоны", "Древние дубы", "Биосферный заповедник"]
  }
]
```

### GET /api/reserves/<id>
Возвращает информацию о конкретном заповеднике по ID.

### GET /api/health
Проверка работоспособности API.
