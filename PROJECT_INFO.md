# Информация о проекте "Экология и мир"

## Описание

Информационный ресурс "Экология и мир" - это веб-приложение, созданное для изучения заповедников и охраняемых природных территорий Республики Беларусь. Проект разработан как индивидуальное задание для практики в вузе.

## Основные особенности

### ✅ Реализованные функции

1. **Главная страница (Hero)**
   - Привлекательный баннер с фоновым изображением
   - Призыв к действию
   - Плавная прокрутка к разделам

2. **Раздел "О нас"**
   - Информация о проекте и его миссии
   - Статистика заповедников
   - Изображение природы

3. **Раздел "Направления деятельности"**
   - Четыре основных направления работы
   - Карточки с описанием функций

4. **Раздел "Заповедники"** (заменяет Portfolio)
   - Интерактивная карта с использованием Leaflet
   - Список всех заповедников Беларуси
   - Детальная информация о каждом заповеднике
   - Модальные окна с подробными данными

5. **Раздел "Контакты"**
   - Контактная форма
   - Информация для связи
   - Изображение

6. **Навигация**
   - Фиксированное меню с плавной прокруткой
   - Адаптивное мобильное меню
   - Кнопка возврата наверх

### ❌ Исключенные разделы

- **Shod** - раздел удален
- **Shortcodes** - раздел удален
- **Portfolio** - заменен на раздел с картами заповедников

## Технический стек

### Frontend
- **React 18** - современный UI фреймворк
- **React Router** - навигация
- **Leaflet & React-Leaflet** - интерактивные карты
- **Axios** - HTTP клиент
- **CSS3** - стилизация с использованием Flexbox и Grid

### Backend
- **Python 3** - язык программирования
- **Flask** - веб-фреймворк
- **Flask-CORS** - обработка CORS запросов

## Структура данных

### Заповедники Беларуси

Проект включает информацию о 6 основных заповедниках:

1. Беловежская пуща
2. Березинский биосферный заповедник
3. Припятский национальный парк
4. Нарочанский национальный парк
5. Браславские озера
6. Полесский радиационно-экологический заповедник

Каждый заповедник содержит:
- Название
- Описание
- Площадь (в гектарах)
- Координаты (широта, долгота)
- Год основания
- Путь к изображению
- Особенности заповедника

## API Endpoints

### GET /api/reserves
Возвращает список всех заповедников

### GET /api/reserves/<id>
Возвращает информацию о конкретном заповеднике

### GET /api/health
Проверка работоспособности API
